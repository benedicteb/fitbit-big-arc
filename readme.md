# fitbit-big-arc

## Commands

```
$ fitbit-gen-second-markers --circles --only-hours --custom-class "hour-marker" 138 142
$ fitbit-gen-second-markers --circles --not-hours --custom-class "minute-marker" 140 142
$ fitbit-gen-second-markers --circles -n 31 --center-x 225 --center-y 150 --custom-class "date-marker" 43 45
$ fitbit-gen-second-markers --circles -n 30 --center-x 75 --center-y 150 --custom-class "step-marker" 43 45
$ fitbit-gen-second-markers --circles -n 30 --center-x 150 --center-y 225 --custom-class "battery-marker" 43 45
```
